# SimpleHNC
Simple Hash value calculater and Checksum Tool

This program is part of Power Tools.

# Supported Algorithm
1. CRC32
2. MD5
3. SHA-1
4. SHA-256
5. SHA-384 *Not fully support*
6. SHA-512 *Not fully support*
